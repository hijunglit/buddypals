# buddypals

/ -> Home
/join -> Join
/login -> Login
/search -> Search user

/users/:id -> See user
/users/logout -> Log Out
/users/edit -> Edit my profile
/users/delete -> Delete my profile

/posts/:id -> See post
/posts/:id/edit -> Edit post
/posts/:id/delete -> Delete post
/posts/upload -> Upload post
/posts/comments -> Comment on a post
